## Project Overview

This is the "Aok" Shopify theme (version 9.0) - a Liquid-based e-commerce theme with extensive customization options. The theme supports multiple layouts, multi-language localization, and various product display styles.

**Theme Author**: shopifytemplate@gmail.com
**Documentation**: https://shopifytemplate.gitbook.io/december-os2.0/

## Architecture

### Directory Structure

- **layout/** (2 files): Base HTML templates
  - `theme.liquid` - Main theme layout wrapper
  - `password.liquid` - Password-protected page layout

- **sections/** (107 files): Reusable, configurable page sections with JSON schemas
  - Examples: `announcement-bar.liquid`, `slideshow-*.liquid`, `product-template-*.liquid`
  - Each section contains Liquid markup and a `{% schema %}` block defining customizable settings

- **snippets/** (154 files): Reusable component partials
  - Examples: `header-*.liquid`, `product-*.liquid`, `icon-*.liquid`, `blog-grid-*.liquid`
  - No schema blocks; these are included via `{% render %}` or `{% include %}`

- **templates/** (~100 files): Page templates (mix of `.liquid` and `.json`)
  - JSON templates define section composition (e.g., `404.json`, `product.*.json`)
  - Multiple template variants exist for products, collections, articles, blogs

- **assets/** (66 files): Static assets
  - JavaScript: `global.js`, `lazysizes.min.js`, `facets.js`, `photoswipe.min.js`, etc.
  - CSS: `theme.css`, component-specific styles (`component-*.css`)
  - Fonts: `BlackseasFree.woff`, `BlackseasFree.woff2`

- **config/** (3 files): Theme configuration
  - `settings_schema.json` - Theme customizer settings structure (container width, typography, colors, features)
  - `settings_data.json` - Current theme settings values (large file, ~113KB)
  - `markets.json` - Market-specific configurations

- **locales/** (11 files): Translation files
  - `en.default.json` - Default English translations
  - Other languages: `lv.json`, `da.json`, `de.json`, `es.json`, `fr.json`, `it.json`, `ja.json`, `nl.json`, `pt-BR.json`, `pt-PT.json`

### Liquid Template Patterns

**Sections with schemas**: Sections are self-contained components with settings defined in JSON schema blocks:

```liquid
{% schema %}
{
  "name": "Section Name",
  "class": "css-class",
  "settings": [
    {
      "type": "text",
      "id": "setting_id",
      "label": "Setting Label",
      "default": "value"
    }
  ]
}
{% endschema %}
```

**Accessing settings**: Section settings are accessed via `section.settings.setting_id`, global theme settings via `settings.setting_id`.

**Snippet inclusion**: Snippets are included with `{% render 'snippet-name', param: value %}` or the older `{% include 'snippet-name' %}`.

### Styling System

The theme uses CSS custom properties (variables) extensively for theming:
- `--color-body-text`, `--color-body-bg`, `--color-primary`, `--color-secondary`
- `--g-font-size`, `--g-font-weight-body`
- Container width and gutter controlled via `settings.container` and `settings.gutter`

## Key Features

- **Multiple slideshow variants**: `slideshow-1`, `slideshow-2`, `slideshow-3` with different layouts
- **Product page layouts**: Multiple template variants (e.g., `product.thumb-right.json`, `product.des-tabcenter.json`)
- **Collection layouts**: Grid and list views with customizable columns
- **Blog layouts**: Various grid styles (`blog-grid-*.liquid`)
- **Localization**: Currency selector, language selector, multi-market support
- **Product features**: Quick view, wishlist, compare, countdown timers, product labels (New, Sale, etc.)
- **Cart**: Drawer-style cart (`cartdraw: true`), upsells, gift wrapping, coupon codes, free shipping bar

## Configuration Files

### settings_schema.json

Defines the theme customizer interface with sections like:
- General settings (container width, gutter, layout types)
- Typography (font families, sizes, weights, line heights)
- Colors (primary, secondary, backgrounds, text)
- Product grid settings (layout, image ratios, labels)
- Cart settings (drawer style, upsells, shipping calculator)
- Header/footer layouts

### settings_data.json

Contains current values for all theme settings. This file is auto-generated by Shopify and should generally not be edited manually.

## Working with This Theme

### Making Changes

1. **Modifying sections**: Edit files in `sections/` directory. Each section's schema controls what appears in the Shopify theme customizer.

2. **Updating snippets**: Edit reusable components in `snippets/`. Changes affect all pages using that snippet.

3. **Styling changes**: Most component styles are in `assets/component-*.css`. Global styles in `assets/theme.css`.

4. **JavaScript functionality**: Main script is `assets/global.js`. Component-specific scripts like `assets/facets.js`, `assets/show-more.js`.

5. **Template structure**: JSON templates in `templates/` define which sections appear on each page type.

### Important Conventions

- **Liquid variable assignment**: Use `{%- liquid %}` blocks for cleaner multi-line variable assignments
- **Color settings**: Colors support both solid and gradient values
- **Responsive design**: Separate mobile settings exist for many features (e.g., `h2_font_size_mobile`)
- **Enable flags**: Most features have `enable_*` settings to toggle functionality
- **Animation classes**: Theme supports `animation="fade-up"` and similar attributes

### Template Naming Patterns

- `product.[variant].json` - Product page variants
- `collection.[variant].json` - Collection page variants
- `blog.[variant].json` - Blog page variants
- `article.[variant].json` - Article page variants
- Context-specific: Files ending in `.context.lv.json` are for Latvian market context
